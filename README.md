# zenglue

A simple C wrapper over [ZenLib](https://github.com/Try/ZenLib) for custom language bindings. It can be used for writing Gothic 1 & 2 asset importers for various 3d tools.
The wrapper interacts directly with the VDFS archives.

## Supported features

- Mesh import
    - Static    
    - Hierarchies
    - Skinned
    - Morphing
- World import
    - Mesh 
    - Vob hierarchy
        - Interactables
        - Containers
    - Waynet
    - Lights
- Materials and textures
- Animations
    - Aliases
    - Events
- Modelscript metadata
